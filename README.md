## Getting Started

### Prerequisites

- Node.js (v16 or higher)
- npm or yarn package manager
- iOS Simulator (for Mac users) or Android Emulator

### Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/Mohanish-25/HealthHive.git
   cd HealthHive
   ```

2. **Install dependencies**
   ```bash
   npm install
   ```

3. **Set up your Database**

    Create your Firebase Project add Authentication and add the google-services.json file in android/app directory
  



4. **Start the development server**
   ```bash
   npx react-native start
   ```
